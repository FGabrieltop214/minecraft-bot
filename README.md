# Minecraft Bot

## Overview  
The Minecraft Bot is an automated bot designed to assist players in various tasks within the Minecraft universe. It enhances the gaming experience by providing functionalities such as auto-mining, automated farming, chat commands, and more.

## Features  
- **Auto-Mining**: The bot can automatically mine resources without player intervention.  
- **Automated Farming**: It can plant and harvest crops at specified intervals.  
- **Chat Commands**: Users can interact with the bot through custom chat commands in the game.  
- **Customizable Settings**: Players can customize the bot's behavior and functionalities based on their needs.

## Installation  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/FGabrieltop214/minecraft-bot.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd minecraft-bot  
   ```  
3. Install the required dependencies:  
   ```bash  
   npm install  
   ```  

## Usage  
1. Start the bot by running:  
   ```bash  
   npm start  
   ```  
2. Configure the settings in the `config.json` file as needed.  
3. Use specified commands to interact with the bot during gameplay.
